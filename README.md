# Produk & Program Mikro Produktif

Halaman informasi resmi produk dan program mikro produktif untuk pelaku UMKM.

## Tentang Halaman Ini

Halaman ini menyediakan direktori tautan resmi produk dan program mikro produktif
yang dapat diakses oleh pelaku usaha mikro, kecil, dan menengah (UMKM).

Seluruh tautan yang tersedia di halaman ini mengarah langsung ke situs resmi
**www.bankmandiri.co.id** dan tidak mengumpulkan data apapun dari pengunjung.

## Kategori Produk & Program

- **Kredit Usaha Mikro (KUM)** — informasi dan layanan KUM
- **KUM Talangan Pembelian (KUMTP)** — program talangan untuk kebutuhan usaha
- **Kredit Usaha Rakyat (KUR)** — akses kredit program pemerintah
- **Livin' Merchant** — solusi digital untuk merchant UMKM
- **Mandiri Agen** — layanan keagenan dan informasi program agen
- **Ekosistem & Kemitraan** — program kemitraan dan ekosistem UMKM
- **Tabungan** — produk tabungan untuk pelaku usaha
- **UMKM & Edukasi** — program edukasi dan pengembangan UMKM
